# Backend Engineer Portfolio

A fast, modern, responsive portfolio website tailored for a Backend Software Engineer with strengths in databases and deployment. Built with plain HTML/CSS/JS for zero-dependency hosting.

## Features

- **Modern Design**: Clean, professional dark theme with gradient accents
- **Fully Responsive**: Optimized for all screen sizes (desktop, tablet, mobile)
- **Performance Optimized**: Fast loading with vanilla JavaScript, no framework overhead
- **Zero Dependencies**: Pure HTML, CSS, and JavaScript - deploy anywhere
- **Backend Focus**: Showcases database expertise, DevOps skills, and deployment capabilities
- **Accessible**: Semantic HTML, ARIA labels, and keyboard navigation support
- **Smooth Animations**: Subtle animations and transitions for a polished user experience

## Structure

- `index.html` - Main HTML structure with semantic markup
- `styles.css` - Modern CSS with responsive design and animations
- `script.js` - JavaScript for interactivity and smooth user experience

## Sections

1. **Hero**: Eye-catching introduction with call-to-action buttons
2. **About**: Professional summary highlighting backend engineering expertise
3. **Skills**: Four categories showcasing technical proficiencies:
   - Databases (PostgreSQL, MySQL, MongoDB, Redis)
   - Deployment & DevOps (Docker, Kubernetes, CI/CD)
   - Backend Development (APIs, Microservices, Message Queues)
   - Tools & Practices (Git, Agile, System Design)
4. **Projects**: Featured projects demonstrating real-world experience
5. **Contact**: Multiple contact options (Email, GitHub, LinkedIn)

## Deployment

This portfolio can be deployed to any static hosting service:

- **GitHub Pages**: Push to a `gh-pages` branch
- **Netlify/Vercel**: Connect your repository for automatic deployments
- **AWS S3**: Upload files to an S3 bucket configured for static hosting
- **Any web server**: Simply copy the files to your web root directory

## Local Development

No build process needed! Just open `index.html` in a browser or use a local server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.

## Customization

To personalize the portfolio:

1. **Update content** in `index.html`:
   - Change the title, subtitle, and about section
   - Add your own projects and skills
   - Update contact information

2. **Modify styling** in `styles.css`:
   - Adjust CSS variables at the top for colors and theme
   - Customize fonts, spacing, and layouts

3. **Enhance functionality** in `script.js`:
   - Add new interactions or animations
   - Integrate analytics or contact forms
